# MiRob CAM

`MiRob CAM` 是一个基于 ESP32 平台的嵌入式相机项目，用于实现本地拍摄、存储与预览能力。项目面向机器人视觉采集场景，强调轻量部署、板载交互和可扩展的模块化结构。

当前仓库包含两个子工程：

- `MiRob_cam_s3`：ESP32-S3 主版本，支持 TFT 实时显示、TF 卡照片/视频保存、SoftAP 预览，以及按键与状态灯交互。
- `MiRob_cam_espcam`：ESP32-CAM 兼容版本，提供精简的拍照与网页预览能力，便于在资源受限硬件上快速落地。

## 项目定位

- 面向低功耗边缘端的图像采集与预览
- 兼顾“本地可见”（屏幕/按键）与“网络可看”（网页流）
- 通过统一的模块划分（`camera` / `storage` / `network` / `http` / `display` / `log`）降低移植与维护成本

## 核心能力概览

- 相机采集：支持多种拍摄模式与参数配置
- 本地存储：将拍摄结果保存到 TF 卡
- 实时预览：支持板载屏幕显示与网页访问
- 设备交互：支持按键控制、串口调试与状态指示
